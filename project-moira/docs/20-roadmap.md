# 20 — Roadmap

## Current focus
**Phase 0: Foundation** — Project scaffolding, docs, design system, and card data model setup.

## Up next
1. **Phase 1: Core Engine** — Card data (Major Arcana first), card/spread services, ReadingContext, shuffle/draw logic
2. **Phase 2: Card Library** — Card browser page, card detail page with upright/reversed toggle, filtering by suit/arcana
3. **Phase 3: Daily Draw** — Single card draw page, card flip animation, interpretation display, export
4. **Phase 4: Full Spreads** — Spread browser, spread reading page with positional layout, step-by-step reveal, synthesis
5. **Phase 5: Export & Polish** — PNG/PDF export, animations (shuffle, deal, glow), responsive refinements
6. **Phase 6: Content Completion** — Write remaining Minor Arcana interpretations, fill in all 10 spreads
7. **Phase 7: Deploy** — VPS setup, nginx config, domain, go live

## Later / ideas
- Reading history (localStorage-based, optional)
- Custom card back design
- Dark/light theme toggle
- Sound design (ambient audio, card flip sounds)
- Additional spreads beyond the initial 10
- Progressive Web App (PWA) for offline use
- Multiple deck support (swap card artwork)

## Explicitly out of scope
- AI/LLM integration
- User accounts or authentication
- Backend API or database
- Social/community features
- Paid subscriptions or e-commerce
- Native mobile apps
- Real-time multiplayer readings
